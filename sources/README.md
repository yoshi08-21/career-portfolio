# sources（経験の素材置き場）

整理前のインプットを置く。**会社／在籍コンテキスト単位**でディレクトリを切る（`sources/<会社>/`）。

- タスクごとの**粗い箇条書きメモ**
- 関連資料（タスクディレクトリ・PR・チケット）への**リンク／場所のメモ**
- 整理資料が無いプロジェクトで、git 履歴などから起こした**棚卸しメモ**
- 「この経験はアピールに使えそう」という気づきの走り書き

ここに溜めたものを元に、`appeals/<会社>/` のアピール文を作る。

## 置き方の目安

- `sources/<会社>/<テーマ>.md` にすると `appeals/<会社>/` と対応づけやすい。
- 機密や個人情報を含むメモは `*.local.md` か `private/` に置く（`.gitignore` で除外済み）。
- 各会社ディレクトリは中身が出来た時点で作る（空ディレクトリは git 管理外。`supernova/` は `.gitkeep` で土台のみ用意済み）。
