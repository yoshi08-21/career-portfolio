# 依頼テンプレ（アピール文を作るとき）

以下を埋めて AI に渡す。最低でも上2つ（資料の場所＋メモ）があれば着手できる。

```
転職アピール文を作ってください。
- タスク資料: <ディレクトリ or PR/チケットのリンク>
- 私のメモ: <箇条書きを貼る、または sources/ のファイル名>
- 用途: 職務経歴書 / ポートフォリオ / 面接口頭（どれか）
- 応募職種: <例: バックエンドエンジニア>
- 定量成果: <数字。無ければ「後で記入」>
- 伏せたい固有名: <社名・パートナー名など。無ければ「特になし＝抽象化で」>
- 特に推したい点: <あれば。気づき/工夫のエピソードなど>
```
