# appeals（アピール文・成果物）

完成版・下書きのアピール文を置く。**会社／在籍コンテキスト単位でディレクトリを切り**（`appeals/<会社>/`）、その中に 1テーマ＝1ファイル（`YYYY-MM-<テーマ>.md`）で置く。

各ファイルは原則として以下を含む（テンプレ: `../templates/appeal-template.md`）。

1. メイン版（職務経歴書・ポートフォリオ向け）
2. 短縮版（履歴書の自己PR欄・1段落）
3. さらに伝わりやすくするための補足
4. 面接での深掘り対策（任意）
5. アピールポイント候補（5軸評価・選択用）

レジュメ完成版（`課題/解決策/成果/関与度/使用技術` × 詳細版/圧縮版）は、完了・転記時に同ファイル内へ追記するか、`../templates/resume-template.md` 形式で整形する。
